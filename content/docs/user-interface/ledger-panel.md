The Ledger panel shows information for about several game systems. It is an UI element of the Map Scene. It is a modal UI element and by default it is not visible.

The ledger contains the following pages:

* List of [Settlements](docs/the-nation/settlements/). Providing an overview of all settlements with following information:
  * Name of settlement
  * Population count
* List of [Markets](docs/trade-system/market/).
  * Name of market (in most cases name of settlement)
  * Sub-list with prices of all resources at market

Other components of the Ledger panel:

* Button to close the Ledger panel
